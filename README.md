# Laboratorio de Ciberseguridad con IA

## 🎯 Objetivo
Este proyecto busca documentar prácticas de ciberseguridad en un entorno controlado de laboratorio, integrando Inteligencia Artificial (IA) en cada etapa.  
El enfoque es **didáctico y reproducible**, pensado para que cualquier persona pueda seguir los pasos y aprender.

## 🛠️ Entorno
- Kali Linux (máquina atacante)
- Windows 10 (máquina víctima)
- Herramientas: Nmap, Metasploit, Wireshark, Hydra, entre otras.

## 📑 Modalidad
1. **Práctica técnica**: ejecución de pruebas en laboratorio.  
2. **Interpretación con IA**: análisis de resultados, explicación clara y generación de informes.  
3. **Documentación paso a paso**: guías reproducibles para otros estudiantes y profesionales.  

## 📂 Estructura del repositorio
- `/labs/` → Documentación de cada práctica.  
- `/scripts/` → Comandos y scripts utilizados.  
- `/reports/` → Informes generados con apoyo de IA.  
- `/guides/` → Guías didácticas para montar y reproducir el laboratorio.  

## 🚀 Primer laboratorio
**Enumeración SMB en Windows 10 vulnerable**  
- Escaneo con Nmap.  
- Resultados interpretados con IA.  
- Informe con recomendaciones de mitigación.

Este repositorio es un recurso abierto y educativo.  
La integración de IA convierte cada práctica en un proyecto completo: técnico, interpretativo y documentado.
