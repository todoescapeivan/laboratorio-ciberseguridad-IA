# Guía de instalación del laboratorio

## Requisitos
- VirtualBox o VMware
- ISO de Kali Linux
- ISO de Windows 10

## Pasos
1. Crear dos máquinas virtuales.
2. Configurar red interna.
3. Instalar Kali y Windows.
4. Verificar conectividad entre ambas.
   
# Guía de instalación para Lab 2

## Requisitos
- Kali Linux
- Servidor vulnerable (DVWA o similar)

## Pasos
1. Instalar DVWA en un servidor web.
2. Configurar red interna entre Kali y DVWA.
3. Verificar acceso desde Kali al servidor web.
4. Ejecutar el script `web_scan.sh` con la IP del servidor.
